# Vigil

Vigil is a governance layer for tool and model actions.

This public repo is the safe, shareable surface:
- what Vigil does
- the current private beta position
- the one-page overview

Current beta shape:
- `normalize` structures a proposal into a deterministic record
- `review` judges it safely with no side effects
- `execute` is intentionally off for now

The one-pager is in [index.html](./index.html).

Private beta:
- API keys issued manually
- direct founder support
- contact: `Discord: Wez Desbar`

This repo does not contain the private server implementation or live deployment state.
